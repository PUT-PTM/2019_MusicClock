# MusicClock

## Overview
Clock with alarm option - now you can be woken up everyday by your favorite music.
## Description

## Tools
8-segment display

2x4 tact switch matrix

RTC Module

SD card reader module
## How to run
Connect your MusicClock to Computer with USB. Clone project and run in Clion.
